# Flutter App — Phase 2

Mobile app for the platform. Built with Flutter targeting iOS and Android.

## Rename the platform

Edit `lib/config/platform_config.dart` — one file changes the name everywhere in the Flutter app.

## Setup

```bash
flutter pub get
flutter run
```

## Status

Scaffolded for Phase 2. Core structure and config in place.
Supabase integration ready — just add credentials to `platform_config.dart`.
